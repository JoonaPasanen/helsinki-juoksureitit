# Helsinki juoksureitit

- Sovelluksessa käyttäjät pystyvät jakamaan juoksureittejä. Reitissä lukee pituus, vaikeustaso ja kuvaus.
- Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.
- Käyttäjä pystyy lisäämään omia reittejä ja muokkaamaan ja poistamaan niitä.
- Käyttäjä näkee sovellukseen lisätyt reitit.
- Käyttäjä pystyy etsimään reittejä hakusanalla.
- Käyttäjäsivu näyttää, montako reittiä käyttäjä on lisännyt ja listan käyttäjän lisäämistä reiteistä.
- Käyttäjä pystyy valitsemaan reitin yhden tai useamman luokittelun (esim. pituus alle 10km, vaikeustaso helppo).
- Käyttäjä pystyy antamaan reitille kommentin ja arvosanan. Reitistä näytetään kommentit ja keskimääräinen arvosana.
